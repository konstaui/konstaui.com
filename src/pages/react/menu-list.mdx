# Menu List React Component

Menu List is an extension of [List View](/react/list). Menu List unlike usual links list is designed to indicate currently active screen (or section) of your app. Think about it like a [Tabbar](/react/tabbar) but in a form of a list.

## Contents

## Menu List Components

There are following components included:

- `MenuList` - menu list component
- `MenuListItem` - menu list item element

## MenuList Props

`MenuList` component doesn't have specific props, but as it extends [`List` component](/react/list), it supports all [`List` props](/react/list#list-props)

## MenuListItem Props

`MenuListItem` component extends [`ListItem` component](/react/list-item), it supports all [`ListItem` props](/react/list-item#listitem-props) and the following additional props:

<MenuListItemProps />
