## Checkboxes List

Checkboxes List is not a separate component, but just a particular case of using [`List`](/svelte/list) and [`ListItem`](/svelte/list-item) components.

```html
<List>
  <!-- Additional "label" prop on ListItem -->
  <ListItem label title="Books">
    {#snippet media()}
    <!-- Pass Checkbox to list item media -->
    <Checkbox component="div" name="my-checkbox" />
    {/snippet}
  </ListItem>
  <ListItem label title="Movies">
    {#snippet media()}
    <!-- Pass Checkbox to list item media -->
    <Checkbox component="div" name="my-checkbox" />
    {/snippet}
  </ListItem>
</List>
```
