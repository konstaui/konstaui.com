# Link Svelte Component

## Contents

Link is main component to create links for navigation, custom actions, switching tabs, open/close modals, etc.

## Link Components

There are following components included:

- `Link`

## Link Props

<LinkProps />

## Examples

#### Navigation Link

```html
<Link href="/about">About</Link>
```

#### External Link

```html
<Link href="https://google.com" target="_blank"> Google </Link>
```

#### Svelte Router Link

```jsx
<Link linkProps={{ to: '/about' }} component={RouterLink}>
  About
</Link>
```

#### Link In Navbar

```jsx
<Navbar title="My App">
  {#snippet right()}
  <Link navbar onClick={openMenu}>
    Menu
  </Link>
  {/snippet}
</Navbar>
```

#### Link In Toolbar

```jsx
<Toolbar>
  <Link toolbar onClick={action1}>
    Action 1
  </Link>
  <Link toolbar onClick={action2}>
    Action 2
  </Link>
</Toolbar>
```
