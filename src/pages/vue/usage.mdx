# Usage

## Contents

After [installation](/vue/installation) process is done we can start using Konsta UI components in our Vue application.

## Components

The process is pretty straightforward and we need to import Konsta UI Vue components from `konsta/vue`:

```html
<template>
  ...
  <k-block>
    <p>This is block with text</p>
  </k-block>
  <k-block class="space-y-4">
    <p>Here comes the button</p>
    <k-button>Action</k-button>
  </k-block>
  ...
</template>
<script>
  /* App.vue */
  import { kBlock, kButton } from 'konsta/vue';

  export {
    components: {
      kBlock,
      kButton,
    },
  }
</script>
```

## App Component

We suppose you use Konsta UI components with some other framework (like [Framework7](https://framework7.io) or [Ionic](https://ionicframework.com)).

But if you use only Konsta UI then we need to wrap all our components with Konsta UI's [App component](/vue/app).

In [App component](/vue/app) we can define global theme (iOS or Material) and other useful globals:

```html
<template>
  <k-app theme="ios">
    <k-page>
      <k-navbar title="My App" />
      <k-block>
        <p>Here comes my app</p>
      </k-block>
    </k-page>
  </k-app>
</template>
<script>
  import { kApp, kPage, kNavbar, kBlock } from 'konsta/vue';

  export default {
    components: {
      kApp,
      kPage,
      kNavbar,
      kBlock,
    },
  };
</script>
```

## Konsta UI Provider

And if you do use Konsta UI with other frameworks, we still can (and should) specify Konsta UI's globals (like theme). For this case we need to use [KonstaProvider](/vue/konsta-provider).

We also need to add `k-ios` or `k-material` class to the app's root element.

```html
<template>
  <!-- Wrap our app with KonstaProvider -->
  <k-provider theme="ios">
    <!-- We add extra `k-ios` class to the app root element -->
    <f7-app theme="ios" class="k-ios">
      <f7-view>
        <f7-page>
          <f7-navbar title="My App" />
          <!-- Konsta UI components -->
          <k-block>
            <p>Here comes my app</p>
          </k-block>
          <k-block class="space-y-4">
            <p>Here comes the button</p>
            <k-button>Action</k-button>
          </k-block>
        </f7-page>
      </f7-view>
    </f7-app>
  </k-provider>
</template>

<script>
  /* App.vue */
  // we use main App and Router components from Framework7
  import { f7App, f7View, f7Page, f7Navbar } from 'framework7-vue';
  // we use KonstaProvider instead
  import { konstaProvider, kBlock, kButton } from 'konsta/vue';

  export default {
    components: {
      f7App,
      f7View,
      f7Page,
      f7Navbar,
      konstaProvider,
      kBlock,
      kButton,
    },
  };
</script>
```
